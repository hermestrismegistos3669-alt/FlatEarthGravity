# Trismegistic Conical Gravity Field

Interaktivní Three.js prezentace konceptuálního modelu gravitace pro plochou Zemi ve tvaru komolého kužele.

## Co model obsahuje

- 3D komolý kužel s nastavitelnou horní a spodní základnou.
- Mírně zakřivený obyvatelný povrch.
- Hustotní vrstvy uvnitř tělesa.
- Zvýrazněný spodní / okrajový stabilizační prstenec.
- Centrální dutinu označenou jako `Ад` v popisu modelu.
- Gravitační vektory ukazující lokální směr tíže.
- Režim řezu pro lepší pohled do konstrukce.

## Spuštění lokálně

Stačí otevřít `index.html` v prohlížeči.

Pokud prohlížeč blokuje ES module importy z lokálního souboru, spusť jednoduchý lokální server:

```bash
python3 -m http.server 8000
```

Pak otevři:

```text
http://localhost:8000
```
